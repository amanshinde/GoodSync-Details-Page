# GoodSync ERP - Modern ERP Solution

A sleek, responsive, and premium landing page for **GoodSync ERP**, a comprehensive Enterprise Resource Planning solution. Built with modern web technologies to provide an exceptional user experience.

## 🚀 Features

- **Dynamic Hero Section**: Engaging animations and clear call-to-action.
- **Comprehensive Modules**: Highlights key ERP features like User Roles, Pricing, and Workflows.
- **Interactive UI**: Smooth transitions powered by Framer Motion.
- **Responsive Design**: Optimized for all devices using Tailwind CSS.
- **Health Check**: Built-in health monitoring component.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS, PostCSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM

## 📦 Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/madhwadevcloud-hash/GoodSync-Details-Page.git
   cd GoodSync-Details-Page
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components/`: Reusable UI components (Navbar, Footer, Hero, etc.).
- `src/assets/`: Static assets and images.
- `src/index.css`: Global styles and Tailwind imports.
